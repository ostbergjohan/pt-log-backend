PT-Log API Backend

Manage and track test projects easily.

This is the backend API for the PT-Log application. It provides RESTful endpoints to manage test projects, logs, and configurations. The frontend application consumes this API to provide a user-friendly interface.

Available Endpoints

Health

  • GET/healthcheck - API health check.

Active Projects

  • GET/populate - List active projects (ARKIVERAD = 0).
  • GET/getData?projekt={projekt} - Retrieve test logs for a project.
  • POST/createProject - Create new project.
  • POST/insert - Insert test log.
  • PUT/updateAnalys - Update analysis for a test.
  • POST/addKonfig - Add pacing configuration.
  • POST/addGenerellKonfig - Add general configuration.
  • DELETE/deleteTest - Delete specific test.

Archived Projects

  • GET/populateArkiverade - List archived projects (ARKIVERAD = 1).
  • POST/arkivera?namn={namn} - Archive a project.
  • POST/restore?namn={namn} - Restore archived project.
  • DELETE/deleteProject - Permanently delete project and all its tests.

Monitoring

  • GET/dbpool - Database connection pool statistics.
