PT-Log API

Manage and track test projects easily.

Available Endpoints

  • GET /healthcheck - Check API status.
  • GET /getData?projekt={projekt} - Get all test logs for a project.
  • GET /populate - List all project names.
  • POST /createProject - Create a new project.
  • POST /insert - Insert a new test log.
  • PUT /updateAnalys - Update the ANALYS field for a test log.
